# 🌸 AI Data Classification Using Decision Tree

## Project Overview

This project demonstrates a basic supervised machine learning classification model using Python and Scikit-learn.

The Iris dataset is loaded, divided into training and testing data, and a Decision Tree Classifier is trained to classify flower species.

This project was developed as part of the **DecodeLabs Artificial Intelligence Internship – Project 2**.

---

## Objective

The objective of this project is to understand the workflow of supervised learning by:

- Loading a dataset
- Splitting the dataset
- Training a classification model
- Making predictions
- Evaluating model accuracy
